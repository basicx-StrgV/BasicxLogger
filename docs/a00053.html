string BasicxLogger.ILogger.logID ( string  message,
bool  verifyMessageID = false 
)

Writes the given message, a message ID and the current time stamp to the log file.

If the log file and/or directory is missing, the method will automatically create them.

Parameters
messageThe message that will be writen to the file
verifyMessageIDSet to true if you want to make sure the message id is unique. If set to true, the loging of the message may take longer an use more ram depending on how big your log file is. When the log file exceeds the length of 1,073,741,823 chars (a little over 1GB file size) the ID will not be verifyed.
Returns
The message ID that was automatically assigned to the message. It can be used to identify a specific message.
